Отсканируйте QR-код
Чтобы настроить Android-устройство (версии Android 7+) в управляемом (корпоративном) режиме:
  1. Выполните сброс к заводским настройкам
  2. Нажмите 7 раз подряд на любом месте стартового экрана (должен появиться скрытый режим сканирования QR-кода)
  3. Следуйте инструкциям устройства: примите условия использования, подключитесь к Wi-Fi, отсканируйте QR-код

Код может содержать также ID устройства в системе Headwind MDM (после настройки его не нужно будет вводить). Для включения ID устройства в QR код, введите его на этом экране и нажмите "Обновить".
